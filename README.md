# App Engine Contact Serve

## Comandos a Ejecutar

Entorno virtual

```sh
  virtualenv --python=/usr/bin/python2.7 venv
```

Instalar dependecias

```sh
  sudo pip install -r requirements.txt -t lib
```

Deploy

```sh
  dev_appserver.py .
```
